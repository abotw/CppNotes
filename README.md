### Tricks

```cpp
(n % 2) // => n 是奇数
!(n % 2) // => n 是偶数
```

```cpp
(!n) // => n == 0
n // => n != 0
```

```cpp
while (n --)
{
    // 循环 n 次
}
```

```cpp
// EOF: End Of File == -1
// EOF == -1
scanf("%d", &n) == -1 // => EOF
scanf("%d", &n) != -1 // => Reading... // => ~scanf("%d", &n)
cin >> x && x
cin >> x, x
```
